2
Chunking
rag.py → chunk_pages()

Each page is independently split into ~500-character chunks with 50-character overlap. Chunking per-page keeps page attribution unambiguous.

Example: Page 1 (~2000 chars)
Chunk 0
0 - 500
Chunk 1
450 - 950
Chunk 2
900 - 1400
Chunk 3
1350 - 1850
Chunk 4
1800 - 2000
= 50-char overlap zone
Why overlap? So context isn't lost at chunk boundaries. If a sentence is split at char 500, the overlap ensures the full sentence appears in at least one chunk.
Chunks per page:
Page 1 → 5 chunks (ID 0-4)Page 6 → 4 chunks (ID 17-20)
Page 2 → 4 chunks (ID 5-8)Page 7 → 4 chunks (ID 21-24)
Page 3 → 3 chunks (ID 9-11)Page 8 → 5 chunks (ID 25-29)
Page 4 → 5 chunks (ID 12-16)Page 9 → 3 chunks (ID 30-32)
Page 10 → 2 chunks (ID 33-34)
Total: ~35 chunks
Each chunk is a dataclass:
Chunk(chunk_id=7, text="Employees are entitled...", page=2)
↓~35 Chunk objects

FAISS Indexing
store.py → VectorStore

store.reset() clears any previous PDF's data, then store.add(vectors, chunks) inserts all vectors and chunk metadata as parallel arrays.

IndexFlatIP = brute-force exact search using inner product. Since embeddings are normalized, inner product equals cosine similarity. Fast and exact for up to ~100k vectors.
Phase 2: Query
POST /query
5
Embed the Question
rag.py → embed_texts()

The user's question is embedded with the same model (all-MiniLM-L6-v2) into the same 384-dim vector space.

"How many days of annual leave do employees get?"
→
[0.032, -0.028, 0.071, ..., 0.019] (384-dim)
↓query vector (384-dim)
6
Vector Search (FAISS)
store.py → VectorStore.search()

FAISS computes cosine similarity between the query vector and all 35 stored vectors. Returns the top-K highest-scoring chunks.

cosine_sim(query, each chunk) → sort → top 3

↓top-3 chunks with scores
7
Build Prompt & Call LLM
main.py → build_user_prompt() + llm.py

The retrieved chunks are formatted into a structured prompt and sent to the configured LLM provider.

System Prompt
You are a careful assistant that answers questions strictly from the provided document context. Rules: - Use ONLY the context below. Do not use outside knowledge. - If the answer is not in the context, say: "I couldn't find that in the document." - Be concise. 1-3 sentences unless the question requires more.
User Prompt (auto-generated)
Context: [Chunk 9 | Page 3] ...entitled to 20 days annual leave per year. New employees receive prorated leave... --- [Chunk 5 | Page 2] Section 1: Leave Policy. Employees are eligible for various types of leave including... --- [Chunk 10 | Page 3] ...carry forward up to 5 days of unused leave to the next calendar year... Question: How many days of annual leave do employees get?
LLM (Groq / OpenAI / Anthropic)
temperature=0.2 · max_tokens=800 · provider set via LLM_PROVIDER env var
↓LLM response text
8

main.py → QueryResponse

The answer and source chunks (with similarity scores) are returned so the caller can verify what the answer is grounded in.
